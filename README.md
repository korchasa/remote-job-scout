```bash

npm install

# Development server
./run start

# Run checks
./run check

# Run tests
./run test
```

### CLI Commands

- `./run check`: Full validation (build + lint + test)

- `./run test integration`: Integration tests
- `./run start`: Development server (client + server)
- `./run stop`: Stop server
